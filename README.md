# BrainSpace: An App Made with Expo

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Primary Goals

1. To utilize the full capabilities of React Native and Expo to create stunning visual representations of data.

2. To provide people with a clear look at the topics that are dominating most of their mental resources.
